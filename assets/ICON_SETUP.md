# アイコンセットアップ

Plasmoはブラウザ拡張機能のアイコンとしてPNG形式の画像を必要とします。

## 必要なアイコンサイズ

以下のサイズのアイコンファイルを `assets/` ディレクトリに配置してください:

- `icon.png` (512x512px推奨) - Plasmoが自動的に他のサイズを生成します

## アイコンの作成方法

### オンラインツールを使用
1. [Figma](https://figma.com) や [Canva](https://canva.com) で512x512pxのアイコンを作成
2. PNG形式でエクスポート
3. `assets/icon.png` として保存

### AIツールを使用
- DALL-E、Midjourney、Stable Diffusionなどで生成
- 512x512pxにリサイズしてPNGで保存

### 簡易的なプレースホルダー
開発中は、任意の512x512pxのPNG画像を配置してください。

## 推奨デザイン
- シンプルで認識しやすいデザイン
- Notionのブランドカラー(#2383E2)を使用
- 書類やフォームをイメージしたアイコン
